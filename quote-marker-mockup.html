为什么你这儿不照搬编号
你没有参考文献列表。每条原文论据本身就是引文、已完整显示在眼前; 点击只是把 PDF 滚过去。编号会指向一个不存在的列表 —— 借了"形",没有"实",所以别扭。

② 推荐方案 —— 整段引文可点(不加任何标记)

既然引文就在眼前,标记不该是"另一个指向别处的指针" —— 引文自己就是可点的目标。 这正是 Zotero 批注侧栏的交互:点批注条目 → 阅读器跳过去。无图标、无编号、无多余文字。

推荐默认态 —— 和普通引用块毫无区别
"Key components of SAM 2 that drive the impressive video object segmentation performance include a large multistage image encoder for frame feature extraction and a memory mechanism that stores memory contexts from past frames to help current frame segmentation."
不醒目、不占地、无图标、无编号 —— 你抱怨过的问题全部不存在。
推荐悬停态 —— 光标变手型,左边框变蓝 + 淡蓝底,角落提示
"Key components of SAM 2 that drive the impressive video object segmentation performance include a large multistage image encoder for frame feature extraction and a memory mechanism that stores memory contexts from past frames to help current frame segmentation."
点击跳到 PDF 原文
hover 态兜底可发现性 —— 鼠标移上就知道能点,点一次就学会。 点击 → PDF 跳到该处并选中高亮。↑ 上面这块和默认态那块都是真能 hover 的,移上去试。

结论

一路试下来,所有"标记"都失败在同一点:它们都在引文旁边多加一个东西,而那个东西要么抢眼、 要么有歧义、要么(编号)指向一个不存在的列表。

正解是不加标记 —— 整段引文本身可点。它对齐 Zotero 批注侧栏的原生交互,一次解决你提的全部问题。 可发现性用 hover 态(手型 + 变色 + tooltip)兜底。
