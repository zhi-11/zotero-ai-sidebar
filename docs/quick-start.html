STEP 01 配置第一个模型预设 Configure your first model preset
没有预设时 sidebar 直接进添加状态。测试连接 通过再保存,免得问到一半才发现 key 错了。 No preset = sidebar drops straight into the form. Always run Test connection before saving — fail loudly here.
  1. 在 Zotero 主窗口右下角条目面板切到 AI 标签,看到"还没有预设"卡片。
  2. 点 + 添加预设,展开预设编辑表单。
  3. 选 Provider(openai / anthropic),填一个好认的名称,粘贴 API Key(默认隐藏,点眼睛切换显示),并填 Base URL(官方端点或 OpenAI 兼容端点都可以)。
  4. 在 Models 区域:从下方 OpenAI 预设模型 网格里点选要用的模型,或者在 输入自定义模型 框里写好后点 添加;每个 chip 都可以单独删除,顶部带边框的是当前默认模型。
  5. 调整 Max tokens(单次输出上限)与 Reasoning Summary(思考长度等级)。
  6. 点 测试连接,等绿勾出现再点 保存。
  7. Open Zotero. The right-side item pane has an AI tab — switch to it; you'll see the "no preset" card.
  8. Click + Add preset to expand the editor.
  9. Pick Provider (openai / anthropic), give it a friendly name, paste your API Key (masked by default — click the eye to reveal), and set Base URL (official or any OpenAI-compatible endpoint).
  10. In Models: click model chips from the OpenAI presets grid, or type into the custom model box and hit Add; each chip is removable, the bordered one is the current default.
  11. Tune Max tokens (per-response cap) and Reasoning Summary level.
  12. Hit Test connection — wait for the green check, then Save.

论文概览

Paper overview

π0.5 是一个把 开放世界泛化 作为一等目标的 VLA 模型 — 而不是只在训练分布附近刷分。

π0.5 is a VLA model that treats open-world generalization as a first-class goal, not in-distribution scores.

一、研究背景与问题

1. Background & problem

  • 大多数 VLA 在训练家庭表现好,新家庭一上就崩。
  • 长链指令(如"打扫卧室")需要正确分解到子动作。
  • Most VLAs collapse when moved to a new home/scene.
  • Long-horizon instructions ("clean the bedroom") must decompose into sub-actions.

二、核心方法

2. Core method

多家庭训练数据 + 推理时语言指令分解 → 子动作 → 控制器。

Multi-home training + inference-time language decomposition → sub-actions → controller.
